# MGA CineAudio Reviewer

ブラウザ内だけで、動画と追加音声（最大 16 本）を同期再生し、波形・スペクトラム・マーカー付きコメントでレビュー共有ができるウェブアプリです。MA・劇伴・音楽など、映像と音声をタイムライン上で聴き比べながら修正指示をやり取りする用途向けです。

## バージョン情報

### v0.09 - 2026年6月10日

- Click メトロノームを追加（C キー・オプションバー Click、Tempo/Sig 入力の BPM/拍子グリッドに同期。Tempo/Sig 表示とは独立、デフォルト OFF、設定は次回起動時に復元）。
- クリック音量をマスター出力 RMS に連動（Analyze 不要）。静かな曲でも楽曲に対して常に大きく聞こえる相対レベルを保つ。
- テンポチェンジ・シーク時の先読み予約を安定化。クリック用 WAV（High/Low）を同梱。

### v0.08 - 2026年6月10日

- Phrase 着色 ON 時、リージョン選択＋Alt+Shift+ホイールでキー ±1 半音（-12〜+12）。リージョン尺不変のタイムストレッチ再生。範囲マーカーと「キーを ±N する」コメントが自動連動。
- Signalsmith Stretch を同梱しオフライン事前レンダリングで音質改善。結合境界での再生チェーン分割と handoff を調整し、キー変更↔未変更の繋がりを改善。
- UI 表記を Pitch → Key に統一（オーバーレイ・トースト・マーカー・使い方・ショートカット）。

### v0.07 - 2026年6月8日

- トランスポート・Markers・S/M など主要ボタンをフラットデザインに統一（暗い内側＋色付き枠、Solo/Mute・Hide の ON 状態は塗りつぶし）。Hide/View の配色を表示状態に合わせて入れ替え。
- ± 音量ショートカット時にファイル名付きトーストを表示。Alt+ホイール音量はリージョン選択時のみ有効にし、リージョン自動分割を廃止。音量ラベルをマーカーコメント風に統一。
- 複数・グループ選択時のスプリットとフェードを全リージョンへ一括適用。リージョン選択枠を 1px に細くし、練習記号の複数トラック横位置ずれを全行オーバーレイで修正。
- トーストを明るめのすりガラス風に変更。Ctrl+←/→ マーカージャンプ時に一覧のオレンジ枠が追従。Shift+↑/↓ で波形縦倍率、Shift+Ctrl+↓ で等倍リセット。横ズームヒントを H-Zoom に統一。
- 拡大中は表示範囲内だけ再生し 99% で停止して開始位置へ戻す。再生中のみプレイヘッド追従スクロールを抑止。マーカー一覧でもズームを有効化し、Musical Grid とリージョン表示を改善。
- Alt+I/O でシークバー位置までフェード設定。三角下端に縦線マーカーを追加。範囲ループのキーボード指定を強化（PgUp/PgDn・キーリピート）。ハイライトを青みの白黒反転オーバーレイに変更。
- Phrase 着色 ON 時のリージョン境界ボンドでフレーズ定義を更新し、アンドゥ復元を修正。元練習番号（origin）表示と保存を廃止。P. Offset に . ショートカットを追加。
- ソロを 1 トラック排他トグルに変更（Alt+S 廃止）。Playback Drift を再生中のみ監視し、停止・シーク時は ---- 表示。色分けを 1f/2f 基準に変更。
- 再生中の ±1f はシーク確定後に停止（キーリピート無効）。キーボードジャンプで一度停止し、シーク確定後に再生再開。±1f 移動時のシークヒントトーストを修正。
- Master Vol リセットと Phrase Offset 切替にオレンジ発光・トーストを追加。小節番号ラベルをレーン高さ非依存の固定値に。練習記号を小節線より前面に表示。

### v0.06 - 2026年6月8日

- ↑↓ ジャンプ時のトーストを改善。リージョンはファイル名、マーカーはコメント（範囲は In/Out 付き）、Grid は Phrase / Tempo/Sig の状態に応じて練習番号・小節番号を表示。P. Offset をセッション保存の対象に追加。
- 波形ズームを 5 段階化し、キーボードショートカットを再割り当て。スクラブとキーボードシークのトーストを修正し、最新操作が優先されるようにした。
- Tempo/Sig ON 時にリージョン内小節番号の表示とジャンプを追加。Shift+上段数字の誤フォールバックを防止。フレーズ定義入力欄末尾で上下キー押下時に ,8 を追加し、新セグメントへ続けて調整可能に。
- マーカー TC の ± 操作をトランスポート秒基準に変更。映像終端以降の移動と TC 表示を修正し、クランプしないフレーム換算で ± 操作に追従。
- 結合境界で後ろリージョンのみフェードイン時に手前ゲインが残り波形が肥大化する不具合を修正。再生後に Shift+←/→ の ±1s/±10s スクラブが効かなくなる不具合を修正。
- ネイティブ動画再生時のレベルメーター・LKFS 不具合を修正。モニタータップの再取得・再試行と VideoAnalyzer 診断ログを追加。
- 使い方の目次を 2 階層化し、閲覧中の章だけ展開する折りたたみとアクセント色リンクを追加。

- マーカー一覧の 1 行目が sticky ヘッダー下に隠れる問題を修正。マーカーの秒ベース保持とマウスドラッグによる細かい位置調整を使い方に明記。
