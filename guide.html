はじめに

このアプリの特色

MGA CineAudio Reviewer は、MA・劇伴・音楽など、映像と音声をタイムライン上で並べ、もしくは聴き比べながら、マーカーや範囲マーカーとそれに付随するコメントで修正指示を共有する用途に特化しています。双方が同じ OS/DAW でなくても、ブラウザだけでレビューの共有が行えます。

レビュー内容の受け渡し方法は、状況に応じて次の 2 通りを使い分けられます。

  • マーカーのみ（軽量） — 相手が同じ動画・同じ追加音声を既に持っているとき。マーカー表（タイムコードとコメント）だけを Slack や Teams 等に貼り付け、チャット上でタイムラインに対するフィードバックを返す運用に適しています。
  • エクスポート・インポート — マーカー情報と動画、追加音声をパッケージに含めて送るため、受け取り側はインポートするだけで、レビューした人の状況を完全に再現でき、フィードバックを確認できます。

制作側・クライアント・作曲家など、立場が違っても「Web ブラウザ＋ファイル」だけでレビューできる点が、従来の DAW プロジェクト共有に比べた利点です。アプリは一瞬で立ち上がります。

このアプリでできること

  • 動画と、DAW 等から書き出した追加音声（最大 3 本）を同時に読み込み、タイムライン上で同期して聴き比べる。
  • 波形上で位置を合わせ、Solo / Mute / フェーダーでミックスを調整する。
  • 気になる箇所にマーカー（点または範囲）と Feedback コメントを付け、Slack 等へコピーして返す。
  • ミックス後の音をスペクトラムとレベルメーターで確認する。
  • 必要に応じて .mgacr にメディア込みで保存し、相手のブラウザで Import して同じレビュー状態を再現する。

画面の構成

画面は縦に 5 つの大きなブロックに分かれています。上から順に触ると、読み込みから再生・マーカー・保存まで自然に進められます。

画面上部 → 下部（左から右の配置も記載）

  1. ① タイトル行 — 画面上端。アプリ名とバージョン表示。その直下にプライバシーに関する注意（赤文字）があります。
  2. ② 映像とマーカー — 横並びの 2 列。
    • 左（広い）：Video パネル — 映像プレビュー、タイムコードオーバーレイ、ファイル名・メタ情報。
    • 右（狭い）：Markers パネル — マーカー一覧表（# / In / Out / Length / Feedback）。
  3. ③ 音声波形エリア — 左サイドバーと右の波形タイムラインの 2 列構成。動画・追加音声の読み込みはここ（波形枠）へのドロップでも行えます。
    • 左サイドバー：各レーンの名前、S（Solo）/ M（Mute）/ ×（クリア）、レベルメーター、音量フェーダー、+ Add Track。
    • 右（広い）：Video Audio と Ex 1〜3 の波形、再生ヘッド（赤い縦線）、マーカー表示、範囲ループのハイライト。
  4. ④ トランスポート — 波形の直下。3 段に分かれた操作バー（時刻・ループ・再生、Video Delay・レビュー入出力、マスター音量・表示下限）。
  5. ⑤ スペクトラムとメーター — トランスポートの下。左にスペクトラム、右にステレオの RMS / ピークメーター。
  6. ⑥ ログ — 画面最下部。読み込み・エクスポート・エラーなどのメッセージ。

狭い画面幅では、② の映像とマーカーは縦に積み重なることがあります。それ以外のブロックの上下順序は変わりません。

操作の流れ

初めて使うときは、画面の上から次の順で進めると迷いにくいです。

  1. ② 映像エリアまたは③ 波形エリアへ動画をドロップして読み込む。必要なら追加音声も同時または後から読み込む。
  2. ② 映像を確認し、④ トランスポートの Play または Space で再生・停止する。
  3. ③ 波形上で位置を合わせ、左サイドバーの Solo / Mute / フェーダー でミックスを聴き比べる。Ex 音源の開始位置はリージョンのドラッグで調整する。
  4. 気になる箇所にマーカー（点または範囲）と Feedback を付ける（② 右の Markers パネルと③ 波形上の表示が連動）。
  5. ⑤ スペクトラム・メーターで全体のバランスを確認する。表示下限は④ で変更。
  6. 共有方法を選ぶ — 相手が同じ素材を持っていれば Copy で Slack 等へ、DAW 環境のない相手や一式渡しには Export Review（.mgacr）。

ファイルの読み込み

読み込みは② 映像エリア（Video パネル）と③ 音声波形エリア（波形表示枠）へのドラッグ＆ドロップで行います。ファイルはブラウザ内だけで処理され、外部へ送信されません。未読み込み時は各エリアに控えめな案内文が表示されます。

まとめて読み込む

  • ② 映像エリアまたは③ 波形エリアへ、動画と追加音声（WAV / FLAC / MP3 等）を一度にドロップします。
  • 種類ごとに自動判別され、動画は② 左の Video パネルへ、音声は空いている Ex レーンへ順に割り当てられます（波形エリアへのまとめドロップでは、追加音声はファイルごとに別トラックへ割り当て）。
  • 映像フレームが空のときは、フレームをクリックしてファイル選択ダイアログを開けます。

個別に読み込む

  • 動画 → ② 左の映像エリア（Video パネル全体）へドロップ。
  • 追加音声 → ③ 音声波形エリア（波形表示枠）へドロップ。空いている Ex レーンに入ります。
  • すでに Ex レーンが表示されているときは、読み込み済みレーンの上にドロップすると、次の空き Ex レーンが開いてそこへ読み込まれます。

トラックの増やし方（+ Add Track）

  • Ex レーンは最初は非表示です。音声を読み込むと自動で表示されます。
  • 手動で次の Ex 枠を出すときは、各レーン左サイドバー下の + Add Track を押します（Video Audio レーンにも同ボタンがあり、次の Ex を表示します）。
  • 音声レーンは最大 4 本（動画内音声の Video Audio ＋ 追加用 Ex 1〜3）。追加できるのは最大 3 トラックです。空きが無いと新しい追加音声は読み込まれません。
  • 動画に音声トラックがない場合、Video Audio レーンは自動で非表示になります（Ex のラベル位置は揃います）。

対応形式

  • 動画：ブラウザが再生できる形式（MP4 / WebM / MOV など）。
  • 追加音声：WAV / FLAC / MP3 / OGG / M4A など。

読み込み中の表示

動画のデコード中は画面中央にローディング表示が出ます。完了まで他の操作は待つか、ログ（⑥）で進捗を確認してください。

映像パネル（Video）

② 左列が映像パネルです。パネル上端に Video ラベルと Clear ボタン、その下にファイル名（未読み込み時は Not Loaded）、さらに下が映像フレームです。未読み込み時はフレーム内に Drop a video, or video and extra audio together と表示されます。

再生とタイムコード

  • 読み込み後、映像の上に HH:MM:SS:FF 形式のタイムコードが重なって表示されます。
  • オーバーレイはドラッグで移動、端をドラッグしてサイズ変更できます。フレーム中央付近ではスナップします。位置はブラウザに保存され、次回も復元されます。
  • パネル直下にはファイル名・メタ情報（FPS など）が表示されます。
  • マーカーの Feedback を入力していると、再生中に映像上へコメントが短時間オーバーレイ表示されます（ポイント用と範囲用で別表示）。

同期の考え方

  • 再生位置の基準は音声（マスター）です。波形上の赤い再生ヘッドと、④ トランスポートの現在時刻が一致します。映像は通常は自然再生に任せ、大きくズレたときだけ補正します（カクつき防止）。
  • ④ トランスポート上段の Playback Drift で映像と音声のズレ（フレーム数）を確認できます。色は安全（通常）／注意（オレンジ）／危険（赤）で変わります。動画読み込み後に表示されます。
  • Ex 音源のタイミングずれは、③ 波形上のリージョン移動で合わせます（通常のレビュー作業）。

Clear

Video パネル右上の Clear で動画だけをアンロードします（追加音声やマーカーは残る場合があります。全体リセットは All Clear）。

マーカー（Markers）

② 右列が Markers パネルです。動画または追加音声を読み込んだあと、レビュー用の印とコメントを付けられます。同じマーカーは③ 波形タイムライン上にも縦線・範囲・ラベルとして表示されます。

パネル上部のボタン

  • Hide / View — 波形上のマーカー表示を隠す／戻す（V と同じ）。マーカーが 1 件以上あるときだけ有効。
  • Copy — マーカー一覧をタブ区切りテキストでクリップボードにコピーします（Length 列は含めません）。Slack・Teams・メールなどへの貼り付けを想定しており、相手が同じ動画・音声を手元で再生できる前提で、タイムライン上のどの位置にどんな Feedback かを返すやり取りに適しています。表の列は # / In / Out / Feedback です。スプレッドシートへ貼る用途にも使えます。
  • Paste — Copy と同じ形式（# / In / Out / Feedback のタブ区切り、Length 列なし）のテキストをクリップボードから貼り付け、現在のマーカー一覧をすべて置き換えます。形式が正しくない場合はダイアログで知らせます。貼り付け前に、マーカーが置き換わることを確認するダイアログが出ます。動画または追加音声を読み込んだあとで利用できます。
  • Clear — すべてのマーカーを削除。

ポイントマーカー（1 時刻）

  • 再生位置で Ins を短く押して離すと、押した時点の位置にポイントが追加されます（押下から離すまで 0.2 秒未満）。
  • 一覧の Feedback 列をクリックしてコメントを入力します。再生中は映像上にコメントが短時間表示されます。

範囲マーカー（In 〜 Out）

  • Ins を 0.2 秒以上押したまま離すと、押した位置から離した位置までの範囲マーカーになります。長押し中は In 待ちのプレビューが表示されます。
  • [ で In（開始）を設定し、続けて ] で Out（終了）を設定します。In 設定中はパネルにヒントが出ます。
  • 波形上のマーカー区間の端（ハンドル）をドラッグして In / Out を調整できます。
  • ③ の Ex リージョンで Alt+ホイールによる音量補正を行うと、そのリージョン幅の範囲マーカーと Feedback（3.0dB 上げる / -3.0 dB 下げる など）が自動で連動します。補正が 0.0 dB に戻ったときは、同じ In／Out の自動付与マーカーを削除します（手動で付けた範囲マーカーと In／Out が一致している場合も同じ行を更新／削除します）。
  • 表の In / Out 列をクリックするとタイムコードを直接編集できます（+ / − でフレーム単位の微調整も可能）。

一覧の操作

  • 行をクリックするとその位置へシークします（再生中はジャンプしません — 停止してから操作してください）。
  • Shift + ↑ / ↓ でマーカー間を移動、Alt + ↑ / ↓ で Feedback 行を移動します。
  • 選択中のマーカーは Del で削除できます（行末の削除ボタンからも可能）。

波形上のマーカー表示が重なる場合は、縦に段を下げてラベルを並べます。

音声波形・ミックス

③ 音声波形エリアは、左のコントロール列と右のタイムライン列で構成されています。上から下へ Video Audio、Ex 1、Ex 2、Ex 3 の順でレーンが並びます（表示されているレーンのみ）。追加音声がまだ無いときは、波形枠の上からおよそ25%の位置に Drop extra audio here と控えめに表示されます（背景色なし）。

左サイドバー（各レーン）

  • 上段：トラック名（Video Audio / Ex 1 Track など）。
  • S（Solo）— そのトラックだけを聴く。他トラックはミュート相当。
  • M（Mute）— そのトラックを消音。
  • × — Ex レーンは非表示＋クリア。Video Audio は非表示にできません（動画内音声がある場合）。
  • 縦長のレベルメーターと dB 表示 — そのトラック単体のレベル。
  • 音量フェーダー — 0 dB 付近を中心に ±24 dB 程度まで調整。横の数値が現在の dB。
  • + Add Track — 次の Ex レーン枠を表示（詳細は「ファイルの読み込み」参照）。

右側の波形タイムライン

  • 各トラックの波形が横方向に並びます。時間軸は動画の尺に合わせて共有されます。
  • 赤い縦線が再生ヘッドです。再生中は通過軌跡が残る表示があります。
  • マウスを乗せると、再生ヘッドとは別の薄い縦線（ホバー位置）が表示されます。
  • クリックでその位置へシークします。
  • ホイールでズーム・横スクロール（Ctrl や Shift 併用で高速化）。+ / − や F（全体表示）も使えます（詳細はショートカット表）。

Ex レーンの再生リージョン

追加音声を読み込むと、波形上に半透明のリージョン（クリップ範囲）が表示されます。

  • リージョンの上をクリックするとその位置へシークします。
  • リージョンをドラッグすると、クリップのタイムライン上の開始位置をずらせます（映像より遅れて入る音源の同期に使います）。
  • リージョン上で Alt+ホイールすると ±1 dB 刻みでそのリージョンの音量補正（中央に赤い dB 表示）。補正のたびに、リージョンの In〜Out に合わせた範囲マーカーが自動で付き、Markers パネルの Feedback に補正値と操作（例: 3.0dB 上げる、-1.0 dB 下げる）が入ります。0.0 dB に戻すとその範囲マーカーは削除されます。同じ範囲のマーカーが既にある場合は In／Out と Feedback を更新します。
  • リージョン情報は Ctrl+C でコピーし、Ctrl+V で同じ Ex レーンの末尾にペーストできます。
  • スプリット（X）— リージョン上にポインタがあるときはカーソル直下で分割。リージョン上にないときはアクティブトラック（トラック名が赤い Ex）をシークバー（再生ヘッド）の位置で分割。
  • 結合（B）— 隣接リージョンの境界上にポインタを置いてから実行。

範囲ループ（波形上）

  • 波形エリアで右ドラッグすると、指定した区間だけをループ再生します（ハイライト表示）。
  • 解除：Esc、右クリック、または別の操作でキャンセル（ショートカット表の Esc 優先順位を参照）。

波形上からのミックス操作

ポインタを置いたレーン（Video / Ex）では S（ソロ）・M（ミュート）・PgUp / PgDn（±1 dB）が使えます。左サイドバーのボタン・フェーダーと同じ効果です。

表示されているレーンの数に合わせて波形パネルの高さが変わります。空の Ex 枠は非表示にできます。

トランスポート（再生コントロール）

④ トランスポートは波形の直下、横長の操作バーです。上から 3 段に分かれています。

上段（再生・全体操作）

  • 現在時刻 − 総尺 — マスター FPS に基づくタイムコード（左端）。
  • Playback Drift — 映像と音声のズレ（フレーム数）。動画読み込み後に表示。
  • Loop — 全体のループ再生のオン／オフ（L）。設定は次回起動時に復元されます。
  • How To / Shortcut — それぞれ本ページとショートカット一覧を新しいウィンドウで開きます。
  • All Clear — 動画・追加音声・マーカー・保存セッションをまとめてクリア（Alt+Shift+Ctrl+Del、Mac では Cmd も可）。押すと再生中ならいったん停止してから確認ダイアログが出ます。
  • Play — 再生／停止（Space）。右端。初回再生時は Video / Markers 行が見える位置へ自動スクロールします。

中段（映像遅延・レビュー入出力）

  • Video Delay — 映像表示を音声より遅らせるフレーム数（0〜99）。レイテンシーの大きい再生環境向けの補正用です（例：仮想音声ミキサー経由で聴いており、映像だけ手前に見える／音が遅れて聞こえる場合）。通常は 0 のまま使ってください。クリップの同期ズレや Ex 音源の合わせ込みには使いません（後者は③ のリージョン移動、前者は Playback Drift の確認が主です）。
  • Include in export — エクスポートに含めるメディア（Video / Ex 1〜3）のチェック。
  • Import Review / Export Review — .mgacr の読み込み・保存。

下段（マスター音量・モニター表示下限）

  • Master Vol — ミックス後のマスター音量（% と dB 表示）。⑤ のレベルメーターでクリップ（0 dB 超え）を検知すると、CLIP PROTECT としてマスター音量が自動で下がります（スライダーと表示も連動）。⑥ ログに CLIP PROTECT と減衰量が出ることがあります。基本的には触らなくて構いません — 通常のレビューでは 100％（0 dB）のまま使い、過大なレベルは各トラックのフェーダーで調整する想定です。意図的に戻すときは Ctrl+Shift+V またはスライダー二重クリックで 100％（0 dB）にリセットできます（Mac では Cmd も可）。自動減衰後の値はブラウザに保存され、次回も復元されます。
  • Spectrum floor / Level meter floor — ⑤ のスペクトラムとレベルメーターの表示下限（dB）。

シーク・ジャンプ（動画読み込み後）

  • 矢印キー：1 フレーム（再生中の ±1 フレームは一度停止してから）
  • Shift + 矢印：1 秒
  • Ctrl + 矢印：5 秒
  • Ctrl + Shift + 矢印：10 秒
  • テンキー 0〜9：尺の 0〜90 % へジャンプ

シーク時には画面中央付近に大きなタイムコード表示（シークフラッシュ）が一瞬出ることがあります。

レビューの保存・共有（.mgacr）

④ トランスポート中段の Import Review / Export Review で、レビュー状態を 1 ファイルにまとめます。マーカーの Copy が「同じ素材を既に持つ相手へコメントだけ渡す」方法なら、.mgacr は「動画・追加音声ごと渡して、相手の環境を問わず同じ聴き比べとマーカーを再現する」方法です。

Include in export でメディアを同梱するため、受け取り側に Nuendo / Pro Tools など同じ DAW がなくても、ブラウザと本アプリだけで映像・音声付きのフィードバックが可能です。作曲家・監督・クライアントなど、制作パイプラインの外側のメンバーへ渡すときに有効です。

ショートカットは Shift+Ctrl+Alt+I（インポート）／Shift+Ctrl+Alt+E（エクスポート）です（Mac では Cmd を Ctrl の代わりに使えます）。

Export Review

  • その左の Include in export で、パッケージに含めるメディアを選びます（Video / Ex 1〜3）。チェックしたトラックの動画・追加音声が .mgacr 内に埋め込まれます。
  • 含まれる主な内容：選択した動画・追加音声、ファイル名、マーカー（In / Out / Feedback）、各トラックのミックス（Solo / Mute / 音量）、Ex リージョンの位置、再生位置、表示設定、Video Delay など。
  • 出力ファイルの拡張子は .mgacr です。ファイル共有サービスやメール等で渡し、相手は Import するだけでレビュー環境を再現できます。

Import Review

  • 受け取った .mgacr を選ぶと、エクスポート時に含めた動画・追加音声と、マーカー・ミックス設定がまとめて復元されます。相手側で同じ素材を別途用意する必要はありません（パッケージにメディアが含まれている場合）。
  • 大きなファイルは読み込みに時間がかかることがあります。⑥ ログ欄に進捗や結果が表示されます。

素材を既に共有済みのチーム内では、マーカー Copy だけで足りることも多く、初回提出や DAW を持たない相手への渡しには .mgacr が向いています。

スペクトラム・レベルメーター

⑤ スペクトラムとメーターはトランスポートの下、横並びのモニター領域です。

  • 左（広い）：スペクトラム — 周波数成分の時間変化。表示下限は④ の Spectrum floor で変更。
  • 右（4 本の縦メーター）：レベルメーター — 左から RMS（L）、ピーク L、ピーク R、RMS（R）。各メーター上のランプがクリップを示します。表示下限は Level meter floor で変更。
  • いずれもマスター出力のミックス後を表示します。特定の 1 トラックだけを計測する用途ではありません（トラック別には③ 各レーン左のメーターを参照）。
  • クリップランプが点灯したら、まず各トラックのフェーダーを下げてください。マスター側は CLIP PROTECT により Master Vol が自動で下がることがあります（④ 下段参照）。Master Vol は基本的に手動では触らなくて構いません。

ログ

⑥ ログは画面最下部のテキスト行です。読み込み完了、エクスポート結果、同期の警告などが時系列で表示されます。うまくいかないときは、まずここを確認してください。

キーボードショートカット

テキスト入力中は多くのショートカットが無効になります（一部 Esc を除く）。Mac では Ctrl の代わりに Cmd も使えます。

データの保存と復元

  • 読み込んだ動画は IndexedDB にコピーされ、次回ページを開いたときに復元されます（F5 リロード時は自動再生しません）。
  • Loop のオン／オフ、タイムコードオーバーレイ位置、レーンの表示状態などは localStorage に保存されます。
  • 元ファイルを DAW 等で書き出し直しても、本アプリは自動では再読み込みしません。最新版を反映するには再度ドロップしてください。
  • ブラウザのストレージ整理で IndexedDB が消えると、前回のセッションは復元できません。

ヒント・注意

  • ⑥ ログに読み込み・エクスポート・同期のメッセージが出ます。トラブル時の第一確認場所です。
  • 長時間・大容量ファイルではメモリ使用量が増え、操作が重くなることがあります。
  • タブをバックグラウンドにすると、メーターや表示の更新が遅れることがあります（ブラウザの仕様）。正確に見たいときはタブを前面にしてください。
  • マーカー一覧の行をクリックしてシークするとき、再生中はジャンプしません。停止してから操作してください。
  • Master Vol はクリップ時に自動で下がります。普段は 100％のまま、音量調整は各トラックのフェーダーで行うのがおすすめです。
  • Video Delay は 0 が標準です。仮想ミキサーなどレイテンシーが大きい環境でのみ、必要なときだけ数フレーム単位で試してください。
  • 映像と音声のズレの確認は Playback Drift が主です。Ex 音源の合わせ込みは波形上のリージョン移動を使います。
