はじめに

このアプリの特色

MGA CineAudio Reviewer は、MA・劇伴・音楽など、映像と音声をタイムライン上で並べ、もしくは聴き比べながら、マーカーや範囲マーカーとそれに付随するコメントで修正指示を共有する用途に特化しています。双方が同じ OS/DAW でなくても、ブラウザだけでレビューの共有が行えます。

レビュー内容の受け渡し方法は、状況に応じて次の2通りを使い分けられます。

  • マーカーのみ（軽量） — 相手が同じ動画・同じ追加音声を既に持っているとき。マーカー表（タイムコードとコメント）だけを Slack や Teams 等に貼り付け、チャット上でタイムラインに対するフィードバックを返す運用に適しています。
  • エクスポート・インポート — マーカー情報と動画、追加音声をパッケージに含めて送るため、受け取り側はインポートするだけで、レビューした人の状況を完全に再現でき、フィードバックを確認できます。

制作側・クライアント・作曲家など、立場が違っても「Web ブラウザ＋ファイル」だけでレビューできる点が、従来の DAW プロジェクト共有に比べた利点です。アプリは一瞬で立ち上がります。

このアプリでできること

  • 動画と、DAW 等から書き出した追加音声（最大16本）を同時に読み込み、タイムライン上で同期して聴き比べる。
  • 波形上で位置を合わせ、Solo / Mute / フェーダーでミックスを調整する。
  • 気になる箇所にマーカー（点または範囲）とコメントを付け、Slack 等へコピーして返す。
  • ミックス後の音をスペクトラムとレベルメーターで確認する。
  • 必要に応じて .mgacr にメディア込みで保存し、相手のブラウザで Import Review（ボタンまたはドロップ）して同じレビュー状態を再現する。

画面の構成

画面は縦に6つの大きなブロックに分かれています。上から順に触ると、読み込みから再生・マーカー・保存まで自然に進められます。

画面上部 → 下部（左から右の配置も記載）

  1. タイトル行 — 画面上端。アプリ名とバージョン表示。その直下にプライバシーに関する注意（赤文字）があります。
  2. 映像とマーカー — 横並びの2列。
    • 左（広い）：Video パネル — 映像プレビュー、タイムコードオーバーレイ、ファイル名・メタ情報。
    • 右（狭い）：Markers パネル — マーカー一覧表（# / In / Out / Length / Comment）と下段の Additional Comments（セッション全体の追加メモ）。
  3. 音声波形エリア — 左サイドバーと右の波形タイムラインの2列構成。ファイルのドロップはページ内どこでも可能で、ここへドロップしたときは波形枠がハイライト表示されます。
    • 左サイドバー：各レーンの名前、S（Solo）/ M（Mute）/ ×（クリア）、レベルメーター、音量フェーダー、+ Add Track。Audio Track には ▲/▼（表示順の入れ替え）もあります。
    • 右（広い）：Video Audio と各 Audio Track の波形、再生ヘッド（赤い縦線）、マーカー表示、範囲ループのハイライト。
  4. 再生・トランスポート操作 — 2か所に分かれます。映像パネル直下の再生バー（Loop / How To / Shortcut / All Clear / Play）と、波形直下のオプションバー（上段: Tempo/Sig・Phrase / 中段: Click・Analyze・R. Offset・レビュー入出力 / 下段: SAVED・Master Vol）。
  5. スペクトラムとメーター — トランスポートの下。左にスペクトラム、右にステレオの RMS / ピークメーター。
  6. ログ — 画面最下部。読み込み・エクスポート・警告・エラーなどのメッセージ。W/E Only / Actions で表示を絞り込み。

狭い画面幅では、映像とマーカーは縦に積み重なることがあります。それ以外のブロックの上下順序は変わりません。

画面レイアウト

パネル配置は固定の標準レイアウト（縦型）のみが初期表示です。画面上部から順に、タイトル行 → 映像+Markers パネル（左に映像とその下の再生バー、右に Markers パネル）→ 音声トラック → オプションバー → スペクトラム/メーター → ログ → リーディング → フッター、と縦に並びます。

H で横型レイアウトにも切り替えられます。横型では1段目に左列（タイトル行・スペクトラム/メーター・再生バー）・映像・Markers パネルを横並びに、2段目に波形、3段目にオプションバー、4段目にログ・リーディング、5段目にフッターを配置します。切替状態は次回起動時に復元されます（Import/Export には含みません）。

画面中央に最大幅 1300px で表示され、上下に余白があります。狭い画面幅では映像とマーカーは縦に積み重なります（「画面の構成」参照）。パネルの位置やサイズをユーザーが変更する機能はありません。

操作の流れ

初めて使うときは、画面の上から次の順で進めると迷いにくいです。

  1. ページ内のどこへでも動画をドロップして読み込む。必要なら追加音声も同時または後から読み込む。
  2. 映像を確認し、トランスポートの Play または Space で再生・停止する。
  3. 気になる箇所にマーカー（点または範囲）とコメントを付ける（右の Markers パネルと波形上の表示が連動）。
  4. 波形上で位置を合わせ、左サイドバーの Solo / Mute / フェーダー でミックスを聴き比べる。追加音声の開始位置はリージョンのドラッグで調整する。
  5. スペクトラム・メーターで全体のバランスを確認する。表示下限はオプションバーで変更。
  6. 共有方法を選ぶ — 相手が同じ素材を持っていれば Copy で Slack 等へ、DAW 環境のない相手や一式渡しには Export Review（.mgacr）。

ファイルの読み込み

読み込みは、ページ内のどこへでも（テキスト入力欄を除く）ファイルをドラッグ＆ドロップできます。拡張子で種類を判別し、動画は Video パネルへ、追加音声は Audio Track へ、.mgacr は Import Review として読み込みます。映像エリア・波形エリアへドロップしたときは、該当箇所がハイライト表示されます。ファイルはブラウザ内だけで処理され、外部へ送信されません。未読み込み時は各エリアに控えめな案内文が表示されます。

まとめて読み込む

  • ページ内の任意の場所へ、動画と追加音声（WAV / FLAC / MP3 等）を一度にドロップします。
  • 種類ごとに自動判別され、動画は左の Video パネルへ、音声は空いている Audio Trackへ順に割り当てられます（まとめドロップでは、追加音声はファイルごとに別トラックへ割り当て。波形エリア全体へ複数ファイルをドロップした場合も同様に1 ファイルずつ空いている Audio Track へ順に入ります）。
  • 映像フレームが空のときは、フレームをクリックしてファイル選択ダイアログを開けます。

個別に読み込む

  • 動画 — 対応拡張子のファイルをページ内へドロップ（先頭1件を Video パネルへ読み込み）。
  • 追加音声 — 対応拡張子のファイルをページ内へドロップ。空いている Audio Track に入ります。
  • 読み込み済みの Audio Track レーン上へドロップすると、同じトラックに別クリップとして追加されます（1トラックあたり複数ファイル可）。

レビューファイル（.mgacr）の読み込み

  • 受け取った .mgacr は、オプションバー中段の Import Review ボタン（Ctrl+Shift+Alt+I）に加え、ページ内のどこへでも（入力欄以外）ドロップして読み込めます（Import Review と同じ処理）。
  • 複数の .mgacr を同時にドロップした場合は、先頭の1件だけがインポートされます。
  • 動画・追加音声などと混在してドロップした場合も、.mgacr のみが Import Review として読み込まれ、他のファイルは無視されます。
  • 復元内容の詳細は「レビューの保存・共有（.mgacr）」の Import Review を参照してください。

トラックの増やし方（+ Add Track）

  • Audio Track は最初は非表示です。音声を読み込むと自動で表示されます。
  • 手動で次の Audio Track 枠を出すときは、各レーン左サイドバー下の + Add Track を押します（Video Audio レーンにも同ボタンがあり、次の Audio Track を表示します）。Ctrl+N でも同じ操作ができます。
  • 追加音声（Audio Track）は3トラック程度までの利用を推奨しています。現状の実装では最大16トラックまで追加可能です（空きが無いと新しい追加音声は読み込まれません）。
  • 動画に音声トラックがない場合、Video Audio レーンは自動で非表示になります（Audio Track のラベル位置は揃います）。

対応形式

  • 動画：ブラウザが再生できる形式（MP4 / WebM / MOV など）。
  • 追加音声：WAV / FLAC / MP3 / OGG / M4A など。WAV / WAVE に埋め込まれたマーカー/リージョンがある場合は、読み込み時に Markers パネルへ自動反映されます（「WAV マーカー/リージョン」参照）。

読み込み中の表示

  • 動画のデコード中は画面中央にローディング表示が出ます。完了まで他の操作は待つか、ログで進捗を確認してください。
  • 追加音声のデコード中は、該当レーンの波形表示エリア上に Now loading と表示されます。

映像パネル（Video）

左列が映像パネルです。パネル上端に Video ラベル、TC Hide / TC View（映像上タイムコードの表示切替）、Clear ボタン、その下にファイル名（未読み込み時は Not Loaded）、さらに下が映像フレームです。未読み込み時はフレーム内に Drop a video, or video and extra audio together と表示されます。

再生とタイムコード

  • 読み込み後、映像の上に HH:MM:SS:FF 形式のタイムコードが重なって表示されます。
  • パネル右上の TC Hide / TC View で、映像上のタイムコードオーバーレイの表示・非表示を切り替えられます。Markers パネルの Hide ボタンと同じ色です。非表示にしてもトランスポートの現在時刻表示は残ります。設定は localStorage に保存され、Export Review / Import Review（.mgacr）にも含まれます。Export WebM でも非表示のときはタイムコードを焼き込みません。
  • オーバーレイはドラッグで移動、端をドラッグしてサイズ変更できます。フレーム中央付近ではスナップします。位置はブラウザに保存され、次回も復元されます。
  • パネル直下にはファイル名・メタ情報（FPS など）が表示されます。
  • コメント欄に入力していると、再生中に映像上へコメントが短時間オーバーレイ表示されます（ポイント用と範囲用で別表示）。

同期の考え方

  • 再生位置の基準は音声（マスター）です。波形上の赤い再生ヘッドと、トランスポートの現在時刻が一致します。映像は通常は自然再生に任せ、大きくズレたときだけ補正します（カクつき防止）。
  • 映像パネル内の Playback Drift で映像と音声のズレ（フレーム数）を確認できます。色は安全（通常）/注意（オレンジ）/危険（赤）で変わります。動画読み込み後に表示されます。
  • 追加音声のタイミングずれは、波形上のリージョン移動で合わせます（通常のレビュー作業）。

Clear

Video パネル右上の Clear で動画だけをアンロードします（追加音声やマーカーは残る場合があります。全体リセットは All Clear）。その左隣の TC Hide / TC View は映像上タイムコード専用で、動画の有無にかかわらず追加音声のみ読み込み中でも利用できます。

マーカー（Markers）

右列が Markers パネルです。動画または追加音声を読み込んだあと、レビュー用の印とコメントを付けられます。同じマーカーは波形タイムライン上にも縦線・範囲・ラベルとして表示されます。

位置の表し方（秒とフレーム）

マーカーの位置は、フレーム番号ではなく秒（トランスポート上の秒）として記録されます。Export Review（.mgacr）やブラウザ内のセッション保存も同じ形式です。画面上の In / Out 列や映像上のタイムコードは、記録した秒から現在の FPS に基づいてフレーム表記（HH:MM:SS:FF）へ変換して表示しています。

  • 表に見える TC と記録した秒は、常に同じ意味のフレーム境界を指すとは限りません。表示 TC は記録秒を最も近いフレームへ丸めた結果です。
  • Ins・[/]・行クリックでの新規配置は、原則としてその時点の再生位置（秒）をそのまま記録します。ただしPhrase 着色（P）ONかつ再生中に Ins だけで確定したポイント位置、および範囲マーカーの In / Out は、Tempo/Sig 定義に基づき4分音符単位へ丸めます（[/]、停止中、Phrase 着色 OFF では丸めません）。フレーム境界に揃える必要がある場合は、後から In / Out 欄で +/− による ±1 フレーム調整を使ってください。
  • 既存マーカーの In / Out をフレーム単位より細かく調整したい場合は、波形上のマーカー（点・範囲の端や帯）または波形上のシークバー付近のマーカーをマウスでドラッグする操作のみが該当します。ポインタ位置から秒へ直接変換して保存します（Alt 押下中は他マーカー・リージョンへのスナップを抑制できます）。TC 欄の +/−（±1 フレーム / ±1 秒）や Paste はフレーム境界を経由するため、このより細かい刻みでは動きません。
  • TC 欄の ±1 フレーム微調整は、いったんフレーム位置へ変換してから再び秒として保存されます。
  • Copy は記録秒から求めたミリ秒表記（00:00:00.000）を出力します。Paste はそのミリ秒 TC を読み取り、現在読み込まれている FPS で最も近いフレーム位置に合わせてから秒として配置します。DAW 側の TC と 1 フレームずれることがあります。
  • 素材の FPS が異なると、同じ記録秒でも画面上のフレーム TC や Length 列のフレーム数が変わることがあります。
  • Length 列の秒/フレーム表示は、記録秒から算出した参考値です（Copy には含まれません）。

パネル上部のボタン

  • Hide / View — 波形上のマーカー表示を隠す/戻す（V と同じ）。マーカーが 1件以上あるときだけ有効。
  • Copy — マーカー一覧をタブ区切りテキストでクリップボードにコピーします（Length 列は含めません）。Slack・Teams・メールなどへの貼り付けを想定しており、相手が同じ動画・音声を手元で再生できる前提で、タイムライン上のどの位置にどんなコメントかを返すやり取りに適しています。表の列は # / In / Out / Comment です。In/Out の時刻は 00:00:00.000 形式（時・分・秒・ミリ秒。ミリ秒は常に3桁）で出力します。画面上のマーカー表はフレーム表記ですが、Copy では一般的な DAW のタイムラインで使われる秒表示（時:分:秒.ミリ秒）に合わせています。DAW 側の位置と照らし合わせやすくするためです。すべてのマーカーで「時」が 00 のときは、コピー時に先頭の 00: を省略します（例: 01:23.456）。Additional Comments に入力があるときは、表の末尾に In/Out が空の行として Feedback 列のみ付けてコピーします。スプレッドシートへ貼る用途にも使えます。
  • Paste — Copy と同じ形式（# / In / Out / Comment のタブ区切り、Length 列なし）のテキストをクリップボードから貼り付け、現在のマーカー一覧をすべて置き換えます。In/Out は DAW などと同じ 00:00:00.000 形式（ミリ秒3桁）を読み取ります。記録形式は秒のため、貼り付け時はミリ秒 TC を現在の FPS で最も近いフレーム位置に合わせてから秒として配置します（上記「位置の表し方」を参照）。Copy で「時」を省略した表（MM:SS.mmm）を貼ると、先頭に 00: を付けて復元します。Additional Comments は表末尾の In/Out が空の行（Feedback 列のテキスト）、従来形式の Additional Comments: … 行、または ---MEMO--- 区切り以降のテキストから復元します。形式が正しくない場合はダイアログで知らせます。貼り付け前に、マーカーが置き換わることを確認するダイアログが出ます。クリップボードが空、またはブラウザから読み取れない場合は、テキストを直接貼れる貼り付けダイアログが開きます（Esc でキャンセル）。動画または追加音声を読み込んだあとで利用できます。
  • Clear — すべてのマーカーを削除。

ポイントマーカー（1時刻）

  • 再生位置で Ins を短く押して離すと、押した時点の位置にポイントが追加されます（押下から離すまで 0.2秒未満）。Phrase 着色（P）ONかつ再生中に確定したときだけ、Tempo/Sig 定義の4分音符グリッドへ丸めます。
  • 一覧の Comment 列をクリックしてコメントを入力します。再生中は映像上にコメントが短時間表示されます。
  • 波形上の点マーカーをダブルクリックすると、その位置を32×で中央表示します。既に32×のときは全体表示（1×）に戻します。シングルクリックではその位置へシークします。

範囲マーカー（In 〜 Out）

  • Ins を 0.2秒以上押したまま離すと、押した位置から離した位置までの範囲マーカーになります。長押し中は In 待ちのプレビューが表示されます。Phrase 着色（P）ONかつ再生中に Ins で In / Out を確定したとき、それぞれの確定時刻を Tempo/Sig 定義の4分音符グリッドへ丸めます（[/] や停止中では丸めません）。
  • [ で In（開始）を設定し、続けて ] で Out（終了）を設定します。In 設定後〜Out 確定前は、パネル上部に Range: press ] for Out とヒントが出ます。
  • 波形上のマーカー区間の端（ハンドル）をドラッグして In / Out を調整できます。帯の中央付近をドラッグすると In / Out を同時に移動します（クリックのみのときはその位置へシーク）。
  • Audio Track 上でリージョンを Ctrl+クリックで選択したうえで Alt+ホイールによる音量補正を行うと、そのリージョン幅の範囲マーカーとコメント列（3.0dB 上げる/-3.0dB 下げる など）が自動で連動します。補正が 0.0dB に戻ったときは、同じ In/Out の自動付与マーカーを削除します（手動で付けた範囲マーカーと In/Out が一致している場合も同じ行を更新/削除します）。
  • Phrase 着色（P）ON でリージョンを選択し Alt+Shift+ホイールでキーを変更すると、同様に範囲マーカーとコメント列（キーを +1 する など）が自動で連動します（-12〜+12 半音）。Key 0 に戻したときは同じ In/Out の自動付与マーカーを削除します。複数選択時はまとめて同じ半音数だけ変更されます。
  • 表の In / Out 列をクリックするとタイムコードを直接編集できます。+ / − で ±1フレーム、Shift++ / − で ±1秒の微調整ができます（Out が空のときは In を起点に Out を設定）。Out 列を編集中に Del で Out をクリアすると範囲マーカーが点マーカーになります。Enter で確定、Esc で編集取消です。
  • 波形上の範囲マーカーをダブルクリックすると、32×で範囲中央を画面中央に表示します。再生位置は範囲の中央へ移動します。既に32×のときは全体表示（1×）に戻します。

Additional Comments（セッション追加メモ）

Markers パネル下段の Additional Comments 欄に、セッション全体向けの自由記述メモを入力できます。動画または追加音声を読み込むと有効になります。マーカー表とは独立して保存され、Copy/Paste および .mgacr の Export/Import に含まれます。入力中は Esc でフォーカスを外せます。Comment 列や Additional Comments 欄の編集中は、T / P など多くのグローバルショートカットが無効になります（Alt+↑/↓ による一覧行移動など、一部は有効）。

一覧の操作

  • Length 列 — 範囲マーカーの長さ（秒/フレーム）。ポイントマーカーは —。Copy には含まれません（画面上の参考表示のみ）。
  • 行をクリックするとその位置へシークします（再生中はジャンプしません — 停止してから操作してください）。
  • Ctrl+←/→ でタイムライン上の停止点（マーカー表示中（Hide でない）＞小節/フレーズ（Tempo/Sig または Phrase 着色。拍線は含まない）＞リージョンの優先順）へ移動します（→＝次、←＝前）。再生中は押下中一度停止し、キーを離すと再生再開します。Alt+↑/↓ で一覧の前後の行へ移動します（↑＝上の行、↓＝下の行）。フォーカスしている列に応じて移動先が変わります。Comment 列では Comment 欄、In 列では In 欄、Out 列では Out TC が確定済みの行の Out 欄へ移動します。パネル外から操作したときは Comment 欄へ移動します。Comment / In / Out 欄の編集中も有効です。
  • 選択中のマーカーは Del で削除できます（行末の削除ボタンからも可能）。

波形上のマーカー表示が重なる場合は、縦に段を下げてラベルを並べます。

波形上の表示

Markers パネルの内容は、音声波形エリア右側のタイムライン（Video Audio と各 Audio Track をまたぐ共有の時間軸）上にも反映されます。Hide / View（V）で、波形上の縦線・範囲帯・コメントラベルと、シークバー付近のマーカー表示をまとめて隠せます。一覧表の行は残ります。

  • ポイントマーカー — タイムライン上に縦線と、コメントがあればラベルが表示されます。線またはラベルをシングルクリックでその位置へシーク、ドラッグで秒単位の位置を微調整できます（「位置の表し方」参照）。
  • 範囲マーカー — In〜Out の区間が半透明の帯として表示されます。端のハンドルをドラッグして In / Out を変更でき、帯の中央付近をドラッグすると In / Out を同時に移動します（クリックのみのときはその位置へシーク）。
  • 再生リージョンとの違い — Audio Track 上の半透明リージョン（クリップの再生範囲）は、追加音声の配置・トリム・ミックス用です（「音声波形・ミックス」の再生リージョンを参照）。範囲マーカーはレビュー用の注釈であり、マスタータイムライン上の In / Out として Markers パネルと連動します。見た目は似ていますが用途が異なります。
  • ラベルの重なり — 近い位置のマーカーラベルは、縦に段を下げて並べます。拡大表示（32×）中は、マーカー一覧で In / Out TC を編集中を除き、再生ヘッドは画面外に出るときだけ左端へ追従します。
  • ジャンプ — Ctrl+←/→ で次/前の停止点へ移動するとき、マーカー表示中（Hide でない）はマーカー位置が最優先されます（小節/フレーズ・リージョンより先）。

WAV マーカー/リージョン（読み込み・書き出し）

Sound Forge 等と互換の cue / LIST(adtl) / plst 形式で、WAV ファイルにマーカー（点）とリージョン（範囲）を埋め込んだり、読み込んだりできます。日本語ラベルは CP932（Shift_JIS）で扱います。

  • 読み込み（インポート） — Audio Track へ WAV / WAVE を読み込んだとき、ファイル内に埋め込まれたマーカー/リージョンがあれば、自動的に Markers パネルと波形上へ反映されます。クリップをタイムライン途中に配置した場合は、その開始位置に合わせてオフセットされます。既にマーカーがあるセッションでは追加（結合）されます。Import Review（.mgacr）やセッション復元時は、パッケージ側のマーカーを優先するため、WAV からの自動取り込みは行いません。
  • 書き出し（エクスポート） — Export Wave で書き出す WAV に、現在のマーカー一覧（点マーカーと範囲マーカー）が同形式で埋め込まれます（「Export WebM / Export Wave」参照）。同一サンプル位置の点マーカー、同一 In/Out の範囲マーカーは、コメントを / で結合して 1 件として書き込みます。
  • DAW とのやり取り — DAW で付けたマーカー付き WAV を Audio Track に読み込んでレビューを続けたり、本アプリで付けたマーカーを Export Wave で DAW に渡したりできます。映像付きレビューでは Export WebM（マーカーコメントの映像焼き込み）や Export Review（.mgacr）も併用できます。

音声波形・ミックス

音声波形エリアは、左のコントロール列と右のタイムライン列で構成されています。上から下へ Video Audio、Audio Track 1、Audio Track 2、… の順でレーンが並びます（表示されているレーンのみ。追加音声は最大16本まで）。追加音声がまだ無いときは、波形枠の上からおよそ25%の位置に Drop extra audio anywhere on the page と控えめに表示されます（背景色なし）。

左サイドバー（各レーン）

  • 上段：トラック名（Video Audio / Audio Track 1 など）。
  • S（Solo）— そのトラックだけを聴く（他トラックはミュート相当）。すでにソロ中なら再度押すとソロを解除します。
  • M（Mute）— そのトラックを消音。
  • × — Audio Track は非表示＋クリア。Video Audio は非表示にできません（動画内音声がある場合）。
  • 縦長のレベルメーターと dB 表示 — そのトラック単体のレベル。
  • 音量フェーダー — 0dB付近を中心に ±24dB程度まで調整。横の数値が現在のdB。
  • + Add Track — 次の Audio Track 枠を表示（詳細は「ファイルの読み込み」参照）。
  • ▲ / ▼ — Audio Track のみ。直上/直下の表示中レーンと、読み込み内容・ミックス設定・波形表示をまとめて入れ替えます（画面上の並びが1段上/下へ移動）。一番上/下のレーンでは該当ボタンは無効です。Video Audio レーンにはありません。順序はセッション保存および .mgacr に含まれます。

右側の波形タイムライン

  • 各トラックの波形が横方向に並びます。時間軸は動画の尺に合わせて共有されます。マーカー（点・範囲）は全レーンをまたぐタイムライン上に表示されます（「波形上の表示」参照）。
  • 赤い縦線が再生ヘッドです。再生中は通過軌跡が残る表示があります。
  • マウスを乗せると、再生ヘッドとは別の薄い縦線（ホバー位置）が表示されます。
  • クリックでその位置へシークします。空のレーン部分をドラッグすると、押している間その位置で再生しながらシーク（スクラブ）できます。
  • ↑/↓ で横倍率を 1 段階変更できます（1×→4×→8×→16×→32×）。波形表示エリア上ではホイール上/下でも同様です。Ctrl+↑ で 32×、Ctrl+↓ で全体表示（1×）に戻します。波形表示エリア上では Ctrl+ホイール上/下でも同様（上＝32×、下＝1×）です。Shift+↑/↓ で縦倍率（振幅の表示）を 1 段階変更できます（1×→2×→4×→8×→16×。1×より下にはなりません）。Shift+Ctrl+↓/↑ で波形トラック自体の高さを変更できます（↓＝拡大、↑＝縮小。100%〜400%、25% 刻み。ユーザー設定として保存され Import Review 後も維持されます）。波形表示エリア上では Shift+Ctrl+ホイール上/下でも同様です。マーカー一覧の TC・コメント編集中は無効です。拡大時は Shift+ホイールで横スクロールできます。再生中もズーム倍率は維持したまま操作できます。拡大中も通常どおりマスタートランスポートが再生を継続します。再生ヘッドは画面外に出るときだけ左端へ追従します（マーカー一覧の In/Out TC 編集中は32×で中央固定）。
  • ダブルクリックでズーム — マーカー以外の波形上ではクリック位置を32×で中央表示します。既に32×のときは全体表示（1×）に戻します。点/範囲マーカー上の動作は「マーカー」を参照してください。
  • Ctrl+←/→ で表示中要素の優先順に従い次/前へ移動します（マーカー表示中（Hide でない）＞小節/フレーズ（Tempo/Sig または Phrase 着色。拍線は含まない）＞リージョン）。時間が進む方向が → で、再生中は押下中一度停止し、キーを離すと再生再開します（詳細は「マーカー」参照）。

Audio Track の再生リージョン

追加音声を読み込むと、波形上に半透明のリージョン（クリップ範囲）が表示されます。

  • リージョンの上をクリックするとその位置へシークします。
  • リージョンをドラッグすると、クリップのタイムライン上の開始位置をずらせます（映像より遅れて入る音源の同期に使います）。他リージョン・動画終端・マーカーへスナップします。リージョン同士が重なると、重なり区間は等パワー・クロスフェードで再生されます。
  • リージョン端のハンドル（ソース In/Out）をドラッグすると、クリップ内の再生範囲を調整できます。隣接リージョンがあるときは境界が1本のスプリットハンドルとして動き、離すと各リージョンに In/Out ハンドルが現れます。Phrase 着色（P）ON で隣接フレーズ（例: A↔B）の境界にあるときは、小節線へスナップし Phrase 入力欄のフレーズ定義を更新します（着色帯上の境界ハンドルと同じ動作）。リージョン境界にポインタを置くと着色帯のハンドルは一時的に無効化され、リージョンハンドルで操作します。同一フレーズ内の結合境界では、ソース上の分割点のみ動きます。Phrase 着色（P）OFF では、タイムライン上でわずかに隙間や重なりがある隣接境界もスプリットハンドル（または境界付近のリージョン本体）でドラッグできます。
  • リージョン左上・右上のFade In / Fade Out 三角を内側へドラッグすると、リージョン端のフェード長を調整できます（曲線表示付き）。Alt+I（Fade In）/Alt+O（Fade Out）でも、シークバー（再生ヘッド）の位置までフェードを伸ばせます（In＝リージョン頭から、Out＝末尾から）。対象は選択中のリージョン、またはポインタ下/アクティブトラック上のシークバー位置のリージョンです。再生中も即座に反映されます。
  • Phrase 着色（P）OFF のとき、Shift+I / Shift+O でリージョンの In / Out を 1 拍ずつ調整できます（クリップ内容は動かさず、In ハンドル/Out ハンドルをドラッグしたのと同じ経路）。移動量は参照位置の Tempo/Sig 定義に基づき、変拍子の各拍長に追従します（Tempo/Sig 未設定時は不可）。対象は選択中のリージョン、またはポインタ下/アクティブトラック上のシークバー位置のリージョンです。隣接リージョンと重なると等パワー・クロスフェードになり、Fade In / Fade Out 三角で表示されます。波形が連続している境界では操作できません。
  • リージョンを Ctrl+クリックで選択したうえで Alt+ホイールすると ±1dB刻みで音量補正（中央に赤いdB表示）。複数選択時はまとめて同じ dB だけ変更されます。補正のたびに、リージョンの In〜Out に合わせた範囲マーカーが自動で付き、Markers パネルのComment列に補正値と操作（例: 3.0dB 上げる、-1.0dB 下げる）が入ります。0.0dBに戻すとその範囲マーカーは削除されます。同じ範囲のマーカーが既にある場合は In/Out とコメント列を更新します。セッション保存・Export/Import Review に含まれます。
  • Phrase 着色（P）ON のとき、リージョンを Ctrl+クリックで選択したうえで Alt+Shift+ホイールすると ±1 半音刻みでキーを変更（-12〜+12。リージョン中央に Key +1 などの表示。リージョン尺は変えずタイムストレッチで再生）。複数選択時はまとめて同じ半音数だけ変更されます。補正のたびに In〜Out の範囲マーカーが自動で付き、Comment 列に キーを +1 する などが入ります。Key 0（変更なし）に戻すとその範囲マーカーは削除されます。セッション保存・Export/Import Review に含まれます。
  • リージョン情報（ソース In/Out・音量補正 dB・キー設定など）は、Ctrl+クリックでリージョンを選択（シアン枠）したうえで Ctrl+C でコピーし、Ctrl+V で同じ Audio Track のコピー元リージョンの直後にペーストできます。直後にちょうど収まる隙間があればそこへ配置し、足りなければ後続リージョンを後ろへずらします。
  • 複数選択 — Ctrl+クリック（Mac は Cmd）でリージョンを追加/解除します。Ctrl+A で対象 Audio Track（アクティブ/ポインタ下）の全リージョンをまとめて選択できます（他トラックの選択は解除。無音スロットは含みません）。隙間なく連続するリージョン（件数不限）を選んで B でもまとめて結合できます。Phrase 着色（P）ON のときは、リージョン間の無音スロットも Ctrl+クリックで選択でき、無音スロット 1 つ＋リージョン 1 つの組み合わせで E による入れ替えが可能です。Ctrl なしでリージョン以外をクリックすると選択は解除されます。すでにグループ化されているリージョンの1つを選ぶと、グループの全メンバー（他の Audio Track を含む）がまとめて選択されます（もう一度選ぶとグループ全体の選択を解除）。選択中はシークバーと同じシアン色のやや太い枠で示されます。
  • グループ化 — 2つ以上選択した状態で G。Audio Track をまたいで可。成功するとメンバー全員の枠が一度黄色に発光します。通常時の見た目は通常リージョンと同じで、ホバー時にグループの全メンバーが黄色枠になります。グループ内のいずれかをドラッグすると、メンバー全体が同じ時間差で平行移動します（開始位置が異なる場合でも、いずれかがタイムライン先頭に達したらそこでグループ全体が止まります）。
  • グループ解除 — グループ内のリージョンを選択して G（グループ化と同じキー）。
  • 入れ替え — Phrase 着色（P）が ON のときだけ、同一 Audio Track 上で2 リージョン、または無音スロット 1 つ＋リージョン 1 つを Ctrl+クリックで選択して E（Exchange の頭文字）を押すと、タイムライン上の順序（フレーズの並び）を入れ替えます（隣接していなくても可）。各リージョンの元の長さを保ち、先の位置に収まらない場合は後続を後ろへずらして空け、短くなった側では余った隙間を詰めます。入れ替え不能なときは赤いトーストで通知されます。
  • スプリット（X）— Tempo/Sig（T）OFF 時：リージョン上にポインタがあるときはカーソル直下で分割。リージョン上にないときはアクティブトラック（トラック名が赤い Audio Track）をシークバー（再生ヘッド）の位置で分割。Tempo/Sig ON 時はフレーズ分割（「トランスポート」参照）。
  • 結合（B — Bond の頭文字）— Tempo/Sig OFF 時：隣接リージョンの境界上にポインタを置く、シークバーを境界上に置く、または同一 Audio Track 上で隙間なく連続するリージョン（件数不限）を Ctrl+クリックで選択してから実行。同一クリップ内で波形が連続している境界（分割直後のような状態）のみ結合できます。リージョンを入れ替えたり重ねてクロスフェードしている場合、Fade In/Out で境界をまたいでいる場合は結合できません。Tempo/Sig ON 時はフレーズ連結。
  • 削除 — Tempo/Sig OFF 時：Ctrl+クリックでリージョンを選択（シアン枠）したうえで Del で削除（複数選択時はまとめて削除。最後の1つを消すとリージョン表示を解除）。Tempo/Sig ON 時はフレーズ削除を先に判定。
  • 元に戻す/やり直し — スプリット・削除・範囲変更などは Ctrl+Z / Ctrl+Shift+Z で履歴をさかのぼれます（回数無制限）。フレーズ定義の履歴が残っているときはそちらを優先します。
  • Esc — ハンドル操作中はドラッグ取消。リージョン・無音スロットの複数選択中は選択だけ解除。それ以外はポインタ下の Audio Track、またはすべての Audio Track のリージョン表示を解除します（ショートカット表の Esc 優先順位を参照）。
  • 各レーン右端付近には、読み込み済みクリップのインテグレーテッド LKFS（ITU-R BS.1770 系）が表示されます（デコード完了後に計測）。

Phrase 着色時のオーバーレイ（リハーサル名・無音スロット）

Phrase 着色（P）が ON で Tempo/Sig・Phrase 定義が有効なとき、Audio Track レーン上にフレーズ枠ごとのリハーサル名が重なって表示されます。各リージョン・無音スロットの左下には、その枠が保持する Tempo/Sig・小節数（例: 140-4/4 · 8）が併記されることがあります。リージョン間の無音スロットは Ctrl+クリックで選択できます。冒頭小節をアウフタクト（弱起）としてリハーサル名を付けない場合は、オプションバー中段の R. Offset（リハーサルマークオフセット）を ON にしてください。

  • リハーサル名（A / B / C …） — フレーズ枠ごとのリハーサル名。Phrase 着色（P）ON では各枠の中央付近に大きく表示されます（入れ替え後は現在の枠位置に合わせて更新されます）。Tempo/Sig・Phrase 定義が有効で Audio Track にリージョンがあるとき、Shift + 対応する英文字（例: Shift+A）でそのリハーサル名の先頭へジャンプできます（Phrase 着色の表示は不要）。
  • Tempo/Sig メタ（左下） — 各リージョン・無音スロットが保持する Tempo/Sig と小節数（音源小節数と枠小節数が異なるときは 16→8 のように表示）。
  • 無音スロット — 隣接リージョンの間に隙間がある区間。Ctrl+クリックで選択できます（シアン枠）。無音スロット 1 つ＋リージョン 1 つを選んで E すると、リージョンをその無音区間へ移動する入れ替えができます。
  • フレーズ境界 — 隣接リージョン境界にポインタを置くと、着色帯上の境界ハンドルは一時的に無効化され、リージョン In/Out・スプリットハンドルでフレーズ境界を調整できます（「フレーズの編集」の境界ドラッグ参照）。
  • R. Offset — 上記の リハーサルマークオフセット。先頭フレーズをアウフタクトとしてリハーサル名なしにし、本編から A を始めます（詳細は「トランスポート」の R. Offset を参照）。

Phrase 着色（P）OFF のときは、リハーサル名オーバーレイと Tempo/Sig メタは非表示です（リージョン操作自体は従来どおり可能）。

範囲ループ（波形上）

  • 波形エリアで右ドラッグすると、指定した区間だけをループ再生します（ハイライト表示）。
  • 再生中に Shift を押し続けると、押した位置を In・離した位置（現在位置）を Out として範囲指定できます（プレビュー表示）。Shift を離すと確定します。他のキーを押すとキャンセルされます。
  • Shift+←/→：1秒の範囲を指定（→＝現在地から1秒先、←＝1秒前から現在地）。同じキーを繰り返すとその方向へ1秒ずつ範囲を拡張（キーリピート可）。
  • Shift+Ctrl+←/→：現在地から Ctrl+←/→ の次ジャンプ先まで。繰り返すと同方向の停止点を追加（キーリピート可。ジャンプ先は状況により異なる）。
  • 解除：Esc、右クリック、または別の操作でキャンセル（ショートカット表の Esc 優先順位を参照）。

波形上からのミックス操作

ポインタを置いたレーン（Video Audio/Audio Track）では S（ソロ）・M（ミュート）・+/−（±1dB、テンキー可）が使えます。左サイドバーのボタン・フェーダーと同じ効果です。S は対象トラックだけをソロにし他トラックのソロを解除します（対象がミュート中なら自動で解除）。すでにソロ中のトラックで再度押すとソロを解除します。Alt+M は読み込み済み全トラックのミュートを解除します。

波形レーンは Tab でフォーカスできます。フォーカス中は Home/End で先頭/末尾へ、←/→ で1フレームシークできます。次の Audio Track 枠を出すショートカット Ctrl+N も使えます（+ Add Track ボタンと同じ）。

表示されているレーンの数に合わせて波形パネルの高さが変わります。空の Audio Track 枠は非表示にできます。

トランスポート（再生コントロール）

再生・トランスポート操作は、現在の UI では次の2ブロックに分かれています。

  • 映像パネル直下の再生バー — Loop / How To / Shortcut / All Clear / Play。
  • 波形直下のオプションバー — 上段: Tempo/Sig・Phrase / 中段: Click・Analyze・R. Offset・レビュー入出力 / 下段: SAVED・Master Vol など。

また、現在時刻・総尺・Playback Drift は再生バーではなく、映像パネル内のステータス行に表示されます。

映像パネル直下（再生バー）

  • Loop — 全体のループ再生のオン/オフ（L）。設定は次回起動時に復元されます。
  • How To/Shortcut — それぞれ本ページとショートカット一覧を新しいウィンドウで開きます。
  • All Clear — 動画・追加音声・マーカー・保存セッションをまとめてクリア（Ctrl+Shift+Alt+Del、Mac では Cmd も可）。押すと再生中ならいったん停止してから確認ダイアログが出ます。
  • Play — 再生/停止（Space）。Ctrl+Space はシークバー位置の1秒前からプリロール再生（1秒未満の場合は現在位置から）。Alt+Enter は直近の再生開始位置から再生し直します（一度も再生していない場合は無効）。右端。初回再生時は Video/Markers 行が見える位置へ自動スクロールします。

現在時刻 − 総尺とPlayback Driftは、映像パネル内の表示を参照してください。

波形直下（オプションバー上段）

  • 現在地（小節・拍） — ---:--:-- 形式で小節位置を表示。
  • Tempo/Sig — 小節・拍グリッド表示（T）とメーター入力。
    • 基本形式 — 120-4/4（BPM-拍子）。
    • 小節ごとの変化 — カンマ区切り（例: 120-4/4,140-3/4）。括弧 (120-4/4,140-3/4) で交互切替。
    • 変拍子（1 小節内） — + で拍子を連結します（例: 120-3/8+5/8 は 3/8 と 5/8 を1 小節とみなす）。現在地（小節・拍）の拍は小節内通算です（上記例では 1〜8 拍）。
    • 拍子の繰り返し（小節ごと） — : で拍子を連結します（例: 120-3/4:5/4 は 3/4 小節と 5/4 小節を交互に繰り返す）。BPM は共通です。
    • + と : の併用 — 1 つの BPM-拍子定義（120-… の 1 要素）内では併用できません。+ は 1 小節内の変拍子、: は小節ごとの繰り返し、いずれか一方のみ使います（例: 120-3/8+5/8,120-3/4:5/4 のようにカンマ区切りで別要素に分けるのは可）。混在した入力（例: 120-3/4+5/4:7/8）は不正とみなされ、Enter 確定時は直前まで有効だった Tempo/Sigに戻ります（Esc も同様）。
    • 入力欄の ↑/↓ — カーソル位置の BPM または拍子（分子・分母）を ±1（Shift+↑/↓ で ±10）。入力欄の末尾では次小節定義 ,120-4/4 を追加（変拍子の完成形 149-3/4+5/4、拍子繰り返しの完成形 120-3/4:5/4 も同様）。+ または : で終わる入力途中だけは次小節ではなく後半拍子を補完（例: 149-3/8+ → ↑ → 149-3/8+4/4、120-3/4: → ↑ → 120-3/4:4/4。BPM はそのまま、後半のデフォルト拍子は 4/4）。
    • Tab — Tempo/Sig 入力欄と Phrase 入力欄を相互に移動（Tempo/Sig → Phrase、Phrase → Tempo/Sig）。
    • Enter で確定、Esc で取消。波形レーンから直接編集するときは Alt+T / Alt+P で各入力欄へフォーカス。
    • 小節番号 — ON 時は Audio Track リージョン内の小節線横に 1 始まりの小節番号が表示され、テンキーまたは Shift+上段数字でその番号へジャンプできます（詳細は「シーク・ジャンプ」参照）。
  • Phrase — フレーズ着色表示（P）とグループ入力（例: 8 小節、1,8 で最初の1小節のみ別色のあと8小節ごと）。入力欄フォーカス時は、カーソル位置の項目を ↑/↓（±1）と Shift+↑/↓（±10）で調整できます。カンマ区切りの末尾（例: 1,8 の最後）にあるときは、上下どちらでも ,8 が追加されます（続けて8小節のフレーズを指定）。波形レーンから直接編集するときは Alt+P で入力欄へフォーカスします。

フレーズの編集（Tempo/Sig 表示中）

Tempo/Sig（T）が ON のとき、X / B / Del は再生リージョンではなくフレーズ定義の操作になります（この3キーでは Phrase 着色（P）は不要）。編集結果は Phrase 入力欄に反映され、セッション保存の対象です。

  • 分割（X）— 小節線でフレーズを2つに分けます。波形上ではポインタ近くの小節線、波形トラック外（再生中も同様）ではシークバーに最も近い小節線を使います。既存のフレーズ境界上では分割できません。
  • 連結（B — Bond の頭文字）— 隣接する2フレーズを1つにします。波形上ではポインタ近くのフレーズ境界、波形トラック外ではシークバーがフレーズ境界上にあるときのみ（フレーズ途中では不可）。
  • 削除（Del）— 対象フレーズを隣接フレーズへ吸収して消します。波形上ではポインタ下のフレーズ、波形トラック外ではシークバー位置のフレーズ。残り1フレーズのときは削除できません。
  • 境界ドラッグ — 上記キーとは別操作で、Phrase 着色（P）が ON のときだけ使えます。着色帯上の境界ハンドル、または隣接フレーズ境界上のリージョン In/Out・スプリットハンドルをドラッグして、隣接フレーズ間の小節数を小節線に沿って調整できます。確定時に Phrase 入力欄へ反映し、リージョンを構成どおりに切り直します。左へドラッグで左フレーズを短く/右フレーズを長くし、さらに左端で左フレーズを右へ吸収。右へドラッグで逆方向の再分配・結合ができます。
  • リージョン入れ替え — 上記とは別操作で、Phrase 着色（P）が ON のときだけ使えます。同一 Audio Track 上で 2 リージョン、または無音スロット 1 つ＋リージョン 1 つを Ctrl+クリックで選択し E（Exchange の頭文字）でフレーズの並びを入れ替えます（隣接していなくても可。元の長さを保ち、必要なら後続をずらして空け、余りは詰め）。
  • 元に戻す/やり直し — 上記および Phrase 入力欄の変更は Ctrl+Z / Ctrl+Shift+Z で履歴をさかのぼれます。フレーズの履歴が空のときは、従来どおりリージョン編集の Undo/Redo になります。

波形直下（オプションバー中段）

  • Click — メトロノームクリック音（C）。Click ON かつ再生中のみ、BPM/拍子グリッド（Tempo/Sig 入力）の拍に同期して鳴ります。Tempo/Sig 表示（T）のオン/オフとは独立です。通常拍子では小節の 1 拍目が高音（High）、それ以外は低音（Low）。変拍子（例: 120-3/8+5/8）では各拍子グループの先頭拍（1 拍目・4 拍目）が高音、それ以外は低音。音量はマスター出力の RMS に連動し、常に楽曲より大きく聞こえるよう調整されます（Analyze 表示は不要。ミックス経路は通りません）。デフォルト OFF。設定は次回起動時に復元されます（Import/Export には含めません）。
  • Analyze — スペクトラム/メーター枠は常時表示です。チェック ON で Live（解析中）、OFF で解析停止（アイドル表示へ戻す）。A でも Live ↔ 解析停止を切替できます。解析停止中も CLIP PROTECT は動作します。
  • R. Offset — リハーサルマークオフセット（Rehearsal Mark Offset）の表示オプション。冒頭の小節をアウフタクト（弱起）として扱い、その部分にはリハーサル名を付けません。ON にすると先頭フレーズはリハーサル名なしとなり、本編の最初のフレーズから A が始まります。R で切り替え。Import/Export Review およびセッション復元に保存されます。
  • Include in export — エクスポートに含めるメディア設定。Video は動画、Audio は追加音声（読み込み済みの全 Audio Track）を一括で切り替えます。
  • Import Review/Export Review — .mgacr の読み込み（ドロップ可）・保存。
  • Export Wave / Export WebM — 動画未読み込み時は Export Wave（レビューミックスを 48 kHz / 24-bit ステレオ WAV で書き出し。マーカー/リージョンを WAV に埋め込み）。動画読み込み後は Export WebM（タイムコードとマーカーコメントを映像へ焼き込み）に切り替わり、ボタンが発光してモード変更を知らせます。動画を削除すると再び Export Wave に戻り、同様に発光します。

波形直下（オプションバー下段）

  • 保存インジケーター（SAVED / SAVING） — オプションバー下段左端の赤い LED 表示です。作業内容がブラウザ内へ自動保存されているかを示します（押すボタンではありません）。SAVED は保存完了、SAVING は保存中です。マーカー・リージョン・トラック順などを編集したあと、リロードやタブを閉じる前に SAVED になるのを待つと安全です。SAVING の最中にリロードすると、保存の途中だった変更は失われ、前回までに保存が完了した状態に戻ることがあります。編集直後でまだ LED が光っていないときも、書き込み前の変更は残らないことがあります。
  • Master Vol — ミックス後のマスター音量（%とdB表示）。横の表示には再生開始からのインテグレーテッド LKFSも併記されます（停止後も最後の値を保持、再再生で計測し直し）。スペクトラム・メーターのレベルメーターでクリップ（0dB超え）を検知すると、CLIP PROTECT としてマスター音量が自動で下がります（スライダーと表示も連動）。ログに CLIP PROTECT と減衰量が出ることがあります。基本的には触らなくて構いません — 通常のレビューでは100%（0dB）のまま使い、過大なレベルは各トラックのフェーダーで調整する想定です。意図的に戻すときは Ctrl+Shift+V またはスライダー二重クリックで100%（0dB）にリセットできます（Mac では Cmd も可）。自動減衰後の値はブラウザに保存され、次回も復元されます。
  • Spectrum floor/Level meter floor — スペクトラムとレベルメーターの表示下限（dB）。

シーク・ジャンプ（動画読み込み後）

  • ←/→：1フレーム（停止中はキーリピート有効。再生中の ±1フレームは一度停止してから、キーリピート無効）
  • PgUp/PgDn：1秒（キーリピート有効。再生中は押下中一度停止し、キーを離すと再生再開）
  • Shift+PgUp/PgDn：10秒（同上）
  • Ctrl+PgUp：冒頭へ（再生中でもシーク後に再生を再開）
  • Ctrl+PgDn：末尾へ（再生中でもシーク後に再生を再開）
  • Ctrl+←/→：タイムライン上の停止点へ移動（マーカー表示中（Hide でない）＞小節/フレーズ（Tempo/Sig または Phrase 着色。拍線は含まない）＞リージョンの優先順。キーリピート有効。再生中は押下中一度停止し、キーを離すと再生再開）
  • 0〜9（上段の数字キー、Shift なし）：尺の0〜90% へジャンプ（9 は 90%）
  • テンキー 0〜9、または Shift + 上段 0〜9：Tempo/Sig（T）ON かつ現在地が Audio Track リージョン内のとき、リージョン内の小節番号（波形上に表示される 1 始まりの番号）へジャンプ。テンキーは Shift 不要（Windows ではテンキーに Shift が届かないため）。数字を続けて入力し、入力が途切れてから約 0.3 秒で確定（例: テンキー 1→3 で 13 小節目）。Tempo/Sig OFF 時にテンキーを押した場合は % ジャンプにフォールバック。再生中はシーク後に再生を再開
  • Shift + A〜Z：Tempo/Sig・Phrase 定義が有効で Audio Track にリージョンがあるとき、リハーサル名（A / B / C …）の先頭へジャンプ（Phrase 着色の表示は不要）

シーク時には画面中央付近に大きなタイムコード表示（シークフラッシュ）が一瞬出ることがあります。

レビューの保存・共有（.mgacr）

オプションバー中段の Import Review/Export Review で、レビュー状態を1ファイルにまとめます。マーカーの Copy が「同じ素材を既に持つ相手へコメントだけ渡す」方法なら、.mgacr は「動画・追加音声ごと渡して、相手の環境を問わず同じ聴き比べとマーカーを再現する」方法です。

Include in export 設定に基づいてメディアを同梱するため、受け取り側に Nuendo / Pro Tools など同じ DAW がなくても、ブラウザと本アプリだけで映像・音声付きのフィードバックが可能です。作曲家・監督・クライアントなど、制作パイプラインの外側のメンバーへ渡すときに有効です。

ショートカットは Ctrl+Shift+Alt+I（インポート）/Ctrl+Shift+Alt+E（エクスポート）です（Mac では Cmd を Ctrl の代わりに使えます）。

Export Review

  • Include in export の Video がオンなら、動画は .mgacr 内に埋め込まれます。
  • Audio がオンなら、追加音声（読み込み済みの全 Audio Track）をすべて .mgacr に同梱します（Mute / Solo 状態は影響しません）。
  • 含まれる主な内容：選択した動画・追加音声、ファイル名、マーカー（In/Out/Comment）、Additional Comments、各トラックのミックス（Solo/Mute/音量）、Audio Track 上のリージョンの位置（リージョンのグループ化情報を含む）、R. Offset（リハーサルマークオフセット/アウフタクト設定）、再生位置、表示設定など。
  • 出力ファイルの拡張子は .mgacr です。ファイル共有サービスやメール等で渡し、相手は Import するだけでレビュー環境を再現できます。

Export WebM / Export Wave

オプションバー中段右端のボタンは、動画の有無でモードが切り替わります。切り替わった直後はボタンが短く発光します。

  • Export Wave（動画未読み込み時）— 現在のレビューミックス（追加音声 Audio Track のミックス結果）を 48 kHz / 24-bit ステレオ WAV で書き出します。通常はオフラインバウンス（高速）で処理し、失敗時は従来の実時間キャプチャへフォールバックします。タイムライン上のマーカー/範囲マーカーは Sound Forge 互換形式（cue / LIST(adtl) / plst）で WAV に埋め込まれます（「WAV マーカー/リージョン」参照）。映像・タイムコード焼き込みはありません。
  • Export WebM（動画読み込み後）— タイムコードとマーカーコメントを映像へ焼き込んだ WebM を実時間で書き出します。
  • Include in export の Audio がオンなら、どちらのモードでも追加音声（読み込み済みの全 Audio Track）をミックス対象に含めます（Mute / Solo 状態は影響しません）。Export WebM では Video もオンが必要です。
  • 書き出し開始前に、波形の横倍率を全体表示（1×）へ戻し、アナライザーを解析停止にします（書き出し前が Live だった場合は完了後に Live へ戻ります。横倍率は 1× のままです）。
  • 書き出し中は操作ロック用オーバーレイが表示され、Esc でキャンセルできます。
  • Export WebM は実時間ベースで進行するため、書き出し時間は尺に応じて長くなります。Export Wave のオフラインバウンスは通常、尺より短い時間で完了します。

Import Review

  • Import Review ボタンで .mgacr を選ぶか、ページ内のどこへでもドロップして読み込みます（「ファイルの読み込み」参照）。
  • 読み込むと、エクスポート時に含めた動画・追加音声と、マーカー・ミックス設定がまとめて復元されます。相手側で同じ素材を別途用意する必要はありません（パッケージにメディアが含まれている場合）。
  • 大きなファイルは読み込みに時間がかかることがあります。ログ欄に進捗や結果が表示されます。

素材を既に共有済みのチーム内では、マーカー Copy だけで足りることも多く、初回提出や DAW を持たない相手への渡しには .mgacr が向いています。

スペクトラム・レベルメーター

スペクトラムとメーターはトランスポートの下、横並びのモニター領域です。

  • 左（広い）：スペクトラム — 周波数成分の時間変化。表示下限は Spectrum floor で変更。
  • 右（4本の縦メーター）：レベルメーター — 左から RMS（L）、ピーク L、ピーク R、RMS（R）。各メーター上のランプがクリップを示します。表示下限は Level meter floor で変更。
  • いずれもマスター出力のミックス後を表示します。特定の1トラックだけを計測する用途ではありません（トラック別には各レーン左のメーターを参照）。
  • クリップランプが点灯したら、まず各トラックのフェーダーを下げてください。マスター側は CLIP PROTECT により Master Vol が自動で下がることがあります（下段参照）。Master Vol は基本的に手動では触らなくて構いません。

ログ

ログは画面最下部のテキスト行です。読み込み完了、エクスポート結果、同期の警告などが時系列で表示されます。重要度に応じて行の色が変わり、ワーニング行の先頭には [Warning]、エラー行には [Error] が付きます（通常の情報行は従来どおり白系）。

ログ欄右下の W/E Only を ON にすると、ワーニングとエラーの行だけが表示されます（フォールバックや軽微な未適用は黄色、保存失敗やデコード失敗などは赤）。Actions を ON にすると、操作の結果（リージョン編集・読み込み完了など）と警告・エラー行だけが表示され、診断用の詳細行や内部手順は非表示のままです。どちらもデフォルトは OFF で、ページを再読み込みすると OFF に戻ります（ユーザー設定には保存しません）。Copy は、フィルタ ON 時は表示されている行だけをコピーします。

調査用の診断ログや波形上のリージョン操作帯表示は F10 の「開発者向け定数」パネルから切り替えます（実行中のみ有効。再読み込みですべて OFF）。診断ログのいずれかが ON の間はログ行数が無制限になり、すべて OFF のときは 500 行上限です。Download（ログ全文をファイル保存）・Clear（ログ消去）も右端にあります。うまくいかないときは、まずここを確認してください。

データの保存と復元

  • 読み込んだ動画はブラウザ内にコピーされ、次回ページを開いたときに復元されます（F5 リロード時は自動再生しません）。マーカー・追加音声・リージョン・ミックスなどもセッションに含まれます。
  • 編集内容は即座ではなく、短い待ち時間のあと自動保存されます。オプションバー下段の SAVED / SAVING インジケーター（「トランスポート」章参照）で状態を確認できます。
  • Loop のオン/オフ、Click、Analyze、タイムコードオーバーレイの表示・位置、レーンの表示状態などは localStorage に保存されます。ログ欄の W/E Only / Actions は保存されず、再読み込みで OFF になります。
  • 元ファイルを DAW 等で書き出し直しても、本アプリは自動では再読み込みしません。最新版を反映するには、ページ内のどこへでも再度ドロップしてください。
  • ブラウザのストレージ整理で保存データが消えると、前回のセッションは復元できません。

ヒント・注意

  • ログに読み込み・エクスポート・同期のメッセージが出ます。トラブル時の第一確認場所です。
  • 長時間・大容量ファイルではメモリ使用量が増え、操作が重くなることがあります。
  • タブをバックグラウンドにすると、メーターや表示の更新が遅れることがあります（ブラウザの仕様）。正確に見たいときはタブを前面にしてください。
  • マーカー一覧の行をクリックしてシークするとき、再生中はジャンプしません。停止してから操作してください。
  • Master Vol はクリップ時に自動で下がります。普段は 100%のまま、音量調整は各トラックのフェーダーで行うのがおすすめです。
  • 映像と音声のズレの確認は Playback Drift が主です。追加音声の合わせ込みは波形上のリージョン移動を使います。
