共通
テキスト入力中は下記の多くが無効 — 入力欄・Feedback 欄・TC 欄など。一部 Esc は有効（優先順位は末尾参照）
Mac（修飾キー） — 表中の Ctrl は Cmd に読み替えてください
Mac（削除キー） — Mac の場合は、表示「delete」のキー（⌫、Backspace 位置）は Backspace として届きます。表中の Del は Windows の Delete キーです。削除系で Del とある操作は、Mac では delete（⌫）キーを使ってください

シーク・ジャンプ（動画読込後）
1フレーム ←／→ 再生中の ±1 フレームは一度停止してから（キーリピート有効）
1秒 Shift+←／→ 再生中でもシーク後に再生を再開
5秒 Ctrl+←／→ —
10秒 Ctrl+Shift+←／→ —
位置ジャンプ 0〜90％ テンキー 0〜9 尺の 0/10〜9/10 へ（9 は 90％）。テンキーのみ
マーカー（動画または追加音声読込後）
In／Out ドラッグ ハンドル操作（波形上） 波形タイムライン上のハンドル
タイムライン上の表示／非表示 V Markers パネルの Hide/View と同じトグル
ポイント追加 Ins（短押し） 押下から離すまで 0.2 秒未満
範囲追加 Ins（長押し） 0.2 秒以上押したまま離すと、押下〜離しの位置で範囲マーカー
マーカーへ移動 Shift+↑／↓ ↑＝次のマーカー（時間が進む方向）、↓＝前のマーカー。再生中はシーク後に再生を再開
Feedback 行の前後へ Alt+↑／↓ ↑＝一覧で上の行、↓＝下の行（Feedback にフォーカス）
選択中マーカー削除 Del マーカー選択時のみ（Del は Ex リージョン削除を先に判定）
マーカー TC 編集（In／Out 列）
±1フレーム +／−（テンキー可） In／Out 欄フォーカス、または Out 列ホバーで Out 対象。Out 空時は In を起点に Out を設定
±1秒 Shift++／−（テンキー可） 同上（Out 空時も In 起点）
Out クリア Del Out 欄フォーカス時のみ（範囲→点）
確定 Enter 編集内容を確定
編集取消 Esc 編集を破棄
波形タイムライン（動画または追加音声読込後）
シーク クリック（波形上） Ex レーンのリージョン上はクリックでシーク・ドラッグで移動（再生リージョン参照）
ズーム ホイール（波形上） 全体表示（1×）からもズームアウト可（最小 0.25×、最大 24×）
高速ズーム（3倍） Ctrl+ホイール（波形上） —
ズーム +／−（テンキー可） マーカーパネル内では無効。Alt 押下中も無効（Ex リージョン音量と競合しないよう）
高速ズーム（3倍） Shift++／−（テンキー可） 同上。Shift または Ctrl 併用でも高速
波形全体表示（ズーム 1×） F ズームアウト後も F で全体表示に戻す
横スクロール Shift+ホイール（波形上） ズームして横スクロール可能なときのみ
高速横スクロール（3倍） Shift+Ctrl+ホイール（波形上） ズーム時のみ
ソロ（ポインタ下の Video／Ex レーン） S 波形レーン上にポインタがあるとき。左サイドバーの S ボタンと同じ
ミュート（ポインタ下の Video／Ex レーン） M 波形レーン上にポインタがあるとき。左サイドバーの M ボタンと同じ
アクティブトラック音量 ±1 dB PgUp／PgDn ポインタ下の Video／Ex レーン（0 dB をまたぐときは一度 0 dB で止まる）
横スクロール PgUp／PgDn ズーム時のみ。上記の音量操作が使えないとき（ポインタがレーン外など）
高速横スクロール（3倍） Shift+PgUp／PgDn ズーム時のみ
波形レーンにフォーカス時
先頭へ Home 波形レーン（Tab でフォーカス可）にフォーカスがあるとき
末尾へ End 同上
1フレーム ←／→ 同上。シーク・ジャンプの 1 フレームと同じ（レーンにフォーカスがあると他のショートカット用にフォーカスが外れる）
範囲ループ（波形上）
範囲指定 右ドラッグ（波形上） 指定区間のみループ再生。全体ループ（L）とは別
解除 Esc ドラッグ中・範囲ループ再生中
解除 右クリック（波形上） ドラッグ中・範囲ループ再生中
再生リージョン（Ex のみ）
スプリット（2 リージョンに分割） X リージョン上にポインタがあるときはカーソル直下で分割。リージョン上にないときはアクティブトラック（トラック名が赤い Ex）をシークバー（再生ヘッド）の位置で分割
結合（隣接リージョンを 1 つに） B 隣接境界上にポインタを置いてから
ソース In／Out・スプリット点 ハンドル操作（Ex レーン） 隣接時は境界を 1 本のスプリットハンドルで移動。離した後は各リージョンに In／Out
クリップ開始位置（タイムライン） ドラッグ（Ex レーン） リージョン上でドラッグ（他リージョン・動画終端・マーカーへスナップ可。重なり可・重なり部は等パワー・クロスフェード）
リージョン音量（補正 dB） Alt+ホイール（Ex レーン） ポインタ下の Ex リージョンを ±1 dB 刻み（上＝上げ・下＝下げ）。0 dB 以外はリージョン中央に赤文字表示。補正のたびにリージョン In〜Out の範囲マーカーを付け／更新し、Feedback に補正 dB と操作（例: 3.0dB 上げる／-3.0 dB 下げる）。0.0 dB に戻すと同範囲の自動マーカーを削除。セッション保存・エクスポート／インポートに含む
リージョン情報をコピー Ctrl+C ポインタ下の Ex リージョン（ソース In／Out・クリップ・リードパッド・音量補正 dB 等）
リージョンをトラック末尾にペースト Ctrl+V コピー元と同じ Ex レーンの末尾へ追加（直前リージョンと重ならない）
同一 Ex レーンへの追加ファイル ドロップ（Ex レーン） 読み込み済み Ex レーン上へドロップすると別クリップとして追加（トラックあたり複数波形ファイル可）
次の Ex トラックを表示 + Add Track 各 Ex レーン下部のボタン。空きトラックが無いときの自動表示はしない
リージョン削除 Del ポインタ下の Ex リージョンを削除（最後の 1 つで全解除）。マーカー削除より先に判定
元に戻す（スプリット・削除・範囲変更など） Ctrl+Z リージョン編集の履歴をさかのぼる（回数無制限）
やり直し Ctrl+Shift+Z 元に戻すで戻した操作を進める
解除 Esc ハンドル操作中はドラッグ取消。それ以外はポインタ下の Ex レーン、または全 Ex のリージョン解除
レビューの保存・共有（.mgacr）
Import Review（インポート） Shift+Ctrl+Alt+I トランスポート中段の Import Review と同じ
Export Review（エクスポート） Shift+Ctrl+Alt+E トランスポート中段の Export Review と同じ
All Clear（全体クリア） Alt+Shift+Ctrl+Del 動画・追加音声・マーカー・保存セッションをまとめて解除（トランスポート上段の All Clear と同じ）
Esc の優先順位（入力欄でも有効なものあり）
1. 範囲マーカー In 確定待ちの取消 Esc Ins 長押しで In プレビュー中など
2. 範囲ループの解除 Esc 右ドラッグ中・範囲ループ再生中
3. 再生リージョン Esc ハンドル操作中はドラッグ取消。それ以外は Ex レーンまたは全 Ex のリージョン解除
4. マーカー TC 欄の編集取消 Esc In／Out 欄を編集中
5. マーカー選択解除 Esc マーカー選択中・Feedback 編集中・マーカーパネル内フォーカス時
